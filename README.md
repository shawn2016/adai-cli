# adai-cli

### 介绍

一个vue，react等框架的脚手架，快速搭建框架demo，不断更新模板中...

### 特性

- 一次全局安装，所有的开发项目都支持，不需要每个项目都安装配置webpack。
- 支持`ES6`
- 支持热替换
- 支持反向代理
- 默认支持`vue2.0`，支持`react`
- 支持全局less变量定义
- 构建UMD模式的代码。
- 只需要配置`hey.js`配置文件即可使用

### 安装

```
npm install -g adai-cli
```

### 使用

##### 选择模板

```
adai init projectName
```

根据自己的需要选择vue，react模板

```
cd vue
npm install
npm start
```
